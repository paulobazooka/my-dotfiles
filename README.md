# Dotfile
My dotfiles config for alacritty, neovim and bashrc
- ~/.config/nvim
- ~/.config/alacritty
- ~/.bashrc

# Instructions
Run commands
```shell
git clone git@github.com:paulobazooka/dotfiles.git ~/.dotfiles

cd ~/.config
ln -s /home/$USER/.dotfiles/alacritty
ln -s /home/$USER/.dotfiles/nvim

cd ~
ln -s /home/$USER/.bashrc
```

### Enjoy
